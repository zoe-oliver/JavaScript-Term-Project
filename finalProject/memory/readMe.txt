Memory Game:
Purpose - copy the computer's sequence for as long as you can!

How to play: Press the buttong to start the Game. 
Once it starts the computer will highlight a square, pay 
attention and click the same square the the computer highlights.
If you choose the correct option then the computer will add to the 
sequence, watch the pattern anf follow it in the same order. This
will continue and the cmputer will add one new square each time
until the player chooses the wrong sequence. 

This is an online memory game based off of the "simon" toy.
